# IOT_Visualize

## Introduction
IOT_Visualize visualizes IoT network traffic, enhancing understanding of IoT environments.

## Features
- Real-time visualization of IoT traffic.
- Packet data and device behavior analysis.

## Getting Started

### Prerequisites
- Node.js
- Vue.js

### Installation
1. `git clone [repo_url]`
2. `npm install`
3. `npm start`
